Tesla Event Data Recorder (EDR) Resources

To access EDR data from a Tesla vehicle you’ll need a Windows computer, appropriate cables and adapters, the Tesla EDR retrieval program and a MyTesla account. See the help page for more information.

Order the cables needed to retrieve EDR data. Note that a USB-to-CAN adapter is also required in order to retrieve EDR data. See the help page for more information.
